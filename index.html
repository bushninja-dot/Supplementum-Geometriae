V
François Viète’s Supplementum Geometriae occupies a unique place in the late Renaissance: geometric construction is being translated into a symbolic, almost algebraic language. This edition is intended as a digital companion — short reads, interactive diagrams, and modern commentary — that keeps the spirit of Viète while making his ideas accessible using contemporary tools.

Visual Demos — what to expect

This edition emphasizes diagram-first explanations. Each key construction from Viète will be shown as:

  1. A short historical statement (paraphrased)
  2. An interactive SVG demo where you can drag points and see relationships
  3. A modern algebraic reinterpretation (concise)

Algebraic lens (short)

After showing the geometry visually, we'll provide a concise algebraic reinterpretation. For many of Viète's constructions the modern translation is straightforward: represent segments as positive real numbers, operations of joining/partitioning as addition/subtraction, and classical mean-proportional constructions as equations involving square roots. Later we can add companion-matrix visualizations for selected constructions.

Historical notes

Each construction will include a short note about Viète's original Latin phrasing and a modern paraphrase. We will avoid long verbatim Latin passages in the main flow; instead a toggle will reveal the original when the reader wants it.

Bite-size tasks — what we'll build first

  1. Segment algebra demo: interactive segment multiplication / composition (starter)
  2. Mean proportional construction: classic two-mean construction with geometric square-root visualization
  3. Quadratic construction: show Viète's approach to solving a quadratic via geometric steps
  4. Polygon proportions: construct regular n-gon relationships and show algebraic perimeter/area expressions
  5. Appendix: side-by-side modern translation / original text

We'll implement these incrementally — each as a small, self-contained interactive demo + explanation. The next message I will implement the first demo ("Segment algebra demo") as a working interactive widget — unless you prefer a different starting choice from the list above.
